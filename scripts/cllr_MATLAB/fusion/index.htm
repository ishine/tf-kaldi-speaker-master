Fusion

A potentially fusible speaker detector is one that takes speech, x, as input and which outputs a real scalar ‘score’ s(x). A fusion of N such detectors is also a function from x to a fused score, namely sfused(x) = f(s1(x),s2(x),¼,sN(x)).

 

There are many different forms that the function f can take. In cases where the function is dependent on parameters, these parameters can be trained on some development data. We give MATLAB code for two examples of simple linear fusion. These differ in the objective function that is used to train the fusion parameters:

 

  • Linear logistic regression fusion:  This uses a logistic regression optimization objective. It simultaneously achieves fusion and calibration.

 

  • Linear minimum-MSE fusion: This uses a mean-squared-error optimization objective. The discrimination performance of the fusion is similar overall (and perhaps better at a specifiable operating point) than the logistic regression fusion. This fusion also calibrates to some extent, but not as well as logistic regression. Of course, MSE-fused scores can be post-calibrated.
