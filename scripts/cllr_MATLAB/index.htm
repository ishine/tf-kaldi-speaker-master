FoCal

Tools for Fusion and Calibration of automatic speaker detection systems

 

This toolkit makes available some MATLAB code that may be useful for researchers in the field of automatic speaker detection, particularly those that participate in the yearly NIST Speaker Recognition Evaluations.

 

These tools may be applicable to many other fields of pattern recognition and machine learning, but prospective users are advised that the current documentation is interspersed with jargon that is specific to speaker detection.

 

Download

All of the code (MATLAB .m-files) is available for download as FoCal.zip (21 kB).

 

Documentation

The functions of the toolkit are introduced in the below-referenced web pages. (The code contains further help in the form of comments.)

 

 

-       Tools for evaluation of the goodness of speaker detection log-likelihood-ratios.

 

-       Tools for calibration (i.e. converting scores to log-likelihood-ratios).

 

-       Tools for fusion of multiple detection scores.
